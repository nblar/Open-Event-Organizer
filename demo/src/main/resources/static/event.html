Course Description:

HTML and CSS are coding languages commonly used in combination to create the front-end interface of websites. HTML is the language used to structure a website, and CSS provides the ability to style it.

This hands-on workshop will provide a basic introduction to HTML and CSS, where students gain practical application building a HTML page and stylizing it.


Agenda

  • Structure of webpages

  • Explanation of HTML & CSS

  • Tools & resources

  • Intro to the terminal & markdown

  • Build your own web page together
